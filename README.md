# EsportsDb


Baza danych przechowująca informacje o graczach i drużynach CS:GO. 

Na ten moment umożliwia m.in: 

- Wyświetlanie historii klubowej zawodnika
- Zmianę statusu zawodnika (aktywny, rezerwowy) i przeniesienie go do innego zespołu
- Wyświetlanie aktualnego składu drużyny oraz średniej wieku graczy
- Dynamiczne aktualizowanie regionu drużyny na podstawie narodowości zawodników.
